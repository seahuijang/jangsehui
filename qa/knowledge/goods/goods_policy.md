# 상품관리 정책서

> 출처: 파트너포털 이용가이드(파트너사용) v2025.03 + [컬리임직원] Partner Portal admin 이용가이드 v2026.02

---

## 1. 상품 공급상태 전이

| 현재 상태 | 트리거 | 전이 상태 |
|---------|-------|---------|
| 상품생성 | 파트너 공급정보 입력 후 [등록] | MD승인대기 |
| MD승인대기 | 컬리 담당자 승인 | 견적서대기 |
| 견적서대기 | 견적서 등록 + 컬리 승인 | 발주가능 |
| 발주가능 | 견적서 유효기간 만료 | 견적서대기 |

---

## 2. 상품관리 기본 정책

- 한글표시사항 이미지: PNG/JPEG만, 10MB 이하
- 유의성질여부: 유리포함 상품만 체크표기
- 컬리냉매제: '필요없음' = 냉매제 포함, '필요' = 미포함 (역직관적)
- 센터항목: CC별 유통가능일수/소비기한일수, 판매가능일수, 판매마감일수 개별 설정

---

## 3. 계약관리 (견적서)

- 견적서 등록 가능 상태: 견적서대기, 발주가능
- 유효기간 겹침 + 동일 구분: 최신 등록 공급가만 유효
- 공급가 입력: 상품 낱개 단위, VAT 제외 금액
- 견적서 구분: 일반(단가) / 이벤트(단가)
- 유효기간 만료 → '기간만료' → 재등록 + 재승인 필요

## 5. 엑셀 등록/수정 8종

1. 상품&공급사 정보 등록 (신상품)
2. 공급정보 추가 (기존 상품에 공급사코드 추가 = 멀티벤더)
3. 상품 마스터정보 수정
4. 상세 입고지정보 수정
5. (상품단위)센터항목 정보 수정
6. 재고 운영 정보 수정
7. 상품 공급 정보 수정
8. (공급사단위)센터항목 정보 수정 (CO권한 전용)

### 센터항목 수정 주의사항

- '상품 마스터정보 수정/상품 공급 정보 수정' 파일 → **모든 CC에 동일 정보로 저장**
- CC별 다르게 하려면 반드시 **센터항목 정보 수정 양식** 사용

---

## 6. 오더유형 설정

- 센터별 셋별/택배/대량 오더유형 개별 설정
- 변경 시 재고운영정보 상세FC 초기화 → 재설정 필수
- 상품 재고정보 화면은 2CC(김포) 값을 디폴트 노출

---

## 7. 출하코드 정책

- 본품: 마스터코드와 동일
- 대품(증정품/이벤트): 본품의 마스터코드 입력
- 대체코드: 본품에만 입력, 중복 불가

---

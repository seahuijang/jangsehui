# 정산관리 정책서

> 출처: 파트너포털 이용가이드(파트너사용) v2025.03 + [컬리임직원] Partner Portal admin 이용가이드 v2026.02

---

## 1. 정산 상태 전이 (파트너사용)

| 현재 상태 | 트리거 | 전이 상태 |
|---------|-------|---------|
| 승인요청 | 파트너 승인 | 승인완료 → 발행대기 → 계산서발행/국세청전송 |
| 승인요청 | 파트너 반려 | 승인반려 (반려사유 필수 256자) |

---

## 2. 정산관리 기본 정책

- 정산 승인 시 '승인완료' + 계산서 '발행대기' → 자동 역발행
- 반려 시 반려사유 필수 (256자)
- 역발행: 익일 국세청 자동 전송
- 정발행: 파트너 직접 계산서 발행 + 증빙 첨부
- 팝빌 연동 필수 (링크아이디: KURLYBILL_PU)
- 대금지급: 매입금액 - 실상계금액 = 지급예정금액 → 별도 조정 → 최종지급금액

---

## 3. Admin 정산관리 전체 프로세스 (11단계)

```
매입조정 → 정산집계 → 정산현황 → 매입금액관리 → 상계금액관리 → 대금지급액계산 → 대금지급현황
                                                                              ↕
                                                              정산문의 ← 공급사정산승인
                                                                              ↕
                                                              공급사대금지급현황 / 공급사정산문의
```

---

## 4. 매입 조정

- CO부서만 대량 엑셀 수정 (최대 1,000건)
- 정산포함 '제외': 특약매입/3PL 등 불필요 SKU
- 집계여부 'Y' → 금액 수정 불가 (정산취소 후 수정)

---

## 5. 정산 집계

- CO(QA) 권한만, 집계기준일+세무구분+공급사코드+마감일+처리자
- 매입 대상 SKU 집계여부='N' + 정산포함='포함' 이어야 정상

---

## 6. 정산 현황

- 처리 가능: 정산취소/승인요청/역발행대기/(정)계산서발행/계산서발행취소
- 정산취소 가능 상태: 집계완료/승인요청/승인반려/승인완료(역발행실패,국세청실패,정발행대기)
- 계산서 발행취소: 승인완료 상태에서 가능, 매입금액 산출 완료 시 불가 → 매입금액 먼저 취소

---

## 7. 매입금액관리

- 매입금액 산출: 당월분만, 공급사 개별 최대 50개
- 매입금액 = 공급가액 + 부가세액 (상계금액 미차감)
- 미처리: AP기간 다름/신규 정산코드 등 → 확인 후 재시도

---

## 8. 상계금액관리

### 8-1. 원천 상계금액 관리

- 엑셀 등록 최대 1,000건, 10자리 상계코드 자동 생성
- 상계회차 1~3, 상계유형 중복 불가

### 8-2. 구매론 관리

- 구매기업(컬리)↔판매기업(공급사) 간 융자 제도
- 1 공급사 + 1 정산월 = 2+ 은행 가능 → 최종 하나의 한도금액 합산
- 차감 시 은행 순서/금액 구분 없이 총액 기준

### 8-3. 이월 상계금액 조정

- 입금요청금액: 마이너스 1 이하 숫자 필수
- 동일 공급사+동일 정산월 중복 등록 불가
- 매월 1일 00시 자동 마감

### 8-4. 마감 후 변경 관리

- 유형: 영업양수도/수정지급/선지급/컬리조정
- 전월 마감 이후 건에 한하여 등록
- 취소: 전월/전전월 정산월만 (3개월 초과 불가)
- '컬리조정' 항목: 공급사에 미노출

---

## 9. 대금지급액 계산

- 매입금액 + 상계금액 + 구매론 → 회차별 최종지급금액 산출
- 당월분만, 전체 또는 개별 공급사 (최대 50개)
- 원천금액 변경 없으면 기존 대금지급액 유지

---

## 10. 대금지급현황

- 지급내역: 매입금액-실상계금액-차기이월+구매론발행+마감후변경 = 지급예정 → 최종지급
- 재확인필요: 원천금액 변경 감지 시 뱃지
- 공급사에 미노출: 구매론발행금액, 마감변경, '컬리조정'

---

## 11. 정산 문의

- CO(QA) 권한만 답변 작성, 등록 후 수정 불가
- 답변 등록 시 공급사에 알림톡 자동 발송
- 첨부파일 1개만, 개인정보 포함 업로드 금지

# 공급사/화주사 관리 정책서

> 출처: [컬리임직원] Partner Portal admin 이용가이드 v2026.02

---

## 1. 공급사관리 정책

| 항목 | 정책 |
|------|------|
| 등록 권한 | MD/AMD/CO만 버튼 노출 |
| 사업자번호 | 휴/폐업 → 등록 불가, 1:1 운영 원칙 |
| 기본값 | AP기간=10일(월3회), 지급일=60일 |
| 팝빌 연동 | 세금계산서 발행 위해 필수 |
| 엑셀 대량 등록/수정 | 별도 지정 담당자만, 최대 500건 |
| 입고시간 대량수정 | 대량수정 권한 별도 지정, 최대 500건 |
| 공급사정보 다운로드 | 망분리 PC 담당자에 한하여 제공 |
| 판매관리 사용여부 | 공급사별 '사용함' 설정, 팀 내 검토절차 필수 |
| 담당자 설정 | 담당MD/담당AMD/발주담당 필수, 공급사 담당자 최소 1명 |

---

## 2. 관리자 권한 구조

```
권한 (최소 단위) < 권한그룹 (권한 묶음) < 계정 (이메일 기반)
* 계정당 최소 1개 기본권한그룹 필수
* 특수권한그룹은 선택적 추가
* Phase01: old-escm + PP 이중등록 / Phase02(예정): SSO(okta) 연동
```

## 3. Admin 전용 메뉴 구조 (파트너에 없는 메뉴)

| 메뉴 | 설명 |
|------|------|
| 공급사관리 | 공급사 등록/수정/입고시간 설정 |
| 화주사관리 | KLS 화주사 정보 등록 |
| 관리자 > 권한관리 | 계정별 권한 설정 |
| 발주관리 V2 (Beta) | 신규 발주 시스템 (2026-03-09 오픈) |
| 판매관리 (컬리 화면) | 발주그룹 생성 + 재고현황 조회 |
| 발주승인 관리 | 과발주 승인 기준 설정 |
| 정산관리 확장 | 매입조정/정산집계/매입금액관리/상계금액관리/대금지급액계산 |

## 4. 화주사관리 정책 (KLS)

| 항목 | 정책 |
|------|------|
| 서비스구분 | FBK / 3PL (한 번 저장 후 수정 불가) |
| 사업자번호 | 인증 실패해도 등록 가능 |
| 파트너 연계코드 | 3P파트너 UUID (컬리몰 입점 시) |
| 담당자 | 대표운영자 포함 총 5명 |
| 화주사 > 공급사 | 1개 이상 필수 등록, 담당자 5명까지, 승인절차 없음 |
| 권한 신청 | 그룹웨어 > 파트너포털 계정/권한 신청서 (기본권한: KLS) |

## 5. 외부 시스템 연동

| 시스템 | 용도 |
|------|------|
| 카카오 알림톡 | 발주생성, 반품요청, 회수지연, 정산승인요청, 프로모션 합의대기 |
| 팝빌(POPBILL) | 세금계산서 역/정발행, 국세청 전송 |
| 컬리 물류센터 | 입고 처리, 반품 회수 |
| 파트너 데이터 서비스 | 유료 판매 데이터 리포트 (Basic 300만원/월, Premium 900만원/월) |

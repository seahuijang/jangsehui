# 반품관리 정책서

> 출처: 파트너포털 이용가이드(파트너사용) v2025.03 + [컬리임직원] Partner Portal admin 이용가이드 v2026.02

---

## 1. 반품 상태 전이

| 현재 상태 | 트리거 | 전이 상태 |
|---------|-------|---------|
| 공급사확인대기 | 확인완료 (회수방문일/연락처 입력) | 회수대기 |
| 공급사확인대기 | 미응답 4일 | 회수대기(자동) |
| 회수대기 | 회수지연 1회 | 회수지연(1회) |
| 회수대기 | 회수지연 2회 | 폐기 (비용 파트너사 부담) |

---

## 2. 반품 처리 프로세스

1. 반품 알림톡 수신
2. 공급사확인대기 조회
3. 확인완료 (회수방문일/연락처 입력)
4. 회수대기
5. 반품 거래명세서 출력
6. 회수 진행

---

## 3. 주요 정책

- 회수지연 2회 → 폐기 (비용 파트너사 부담)
- 반품 거래명세서 출력 → 회수 진행
- 미응답 4일 경과 시 자동으로 회수대기 전환

---

|--------|------|---------|
| 반품 수거준비 전환 | 반품 상태 → 수거준비 상태 전환 | 일배치 |
